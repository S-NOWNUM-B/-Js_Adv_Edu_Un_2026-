## О проекте

> **Коллекция лабораторных работ для освоения продвинутых возможностей JavaScript и экосистемы Frontend разработки**

Этот репозиторий содержит лабораторные работы различной сложности для углубленного изучения JavaScript: замыкания, прототипы, асинхронность, работа с модулями и паттернами проектирования. Каждая лабораторная работа включает практические задания, которые помогают освоить ключевые концепции и паттерны разработки в современных веб‑приложениях.

## Быстрый старт

### Клонирование репозитория

```bash
git clone <repository-url>
cd -JSADVANCED_Edu_2026-
```

### Установка зависимостей для лабораторной работы

```bash
# Установить все зависимости для всех проектов
pnpm install

# Запустить dev-сервер для лабораторной *.*
pnpm dev:*.*

# Собрать все проекты
pnpm build:all

# Очистка node_moduls
pnpm clean
```
