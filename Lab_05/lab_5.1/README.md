# Lab 5.1: Code Modernization with ES6+ Classes

## Обзор
Этап первый лабораторной работы № 5 посвящен модернизации устаревшего JavaScript-кода путем преобразования функций-конструкторов в современные ES6+ классы, замены var на const/let и внедрения приватных полей для инкапсуляции данных.

## Цель работы
Освоить синтаксис ES6+ классов, приватные поля и методы, а также разработать архитектуру приложения с правильной инкапсуляцией данных и использованием модульной системы.

## Структура проекта

### Файлы в `modernized/`:
- **`core.mjs`** — Основные классы приложения:
  - `Product` — Класс для представления товара (id, name, price)
  - `Cart` — Класс для управления корзиной (addItem, getSubtotal, getTax, getTotal)
  
- **`config.mjs`** — Конфигурационный файл с констатами приложения:
  - `taxRate` — Налоговая ставка
  - Прочие параметры системы
  
- **`utils.mjs`** — Утилиты для работы с данными:
  - `formatPrice()` — Форматирование цены в валютный формат (USD)

- **`main.mjs`** — Точка входа (entry point):
  - Создание товара и корзины
  - Демонстрация работы системы
  - Вывод результата в консоль

### Исходный анализ legacy-кода:
- Использование `var` (риск hoisting и загрязнение scope)
- Функции-конструкторы вместо классов (менее читаемо)
- Отсутствие инкапсуляции приватных данных
- Конкатенация строк вместо шаблонных литералов
- Common JS или смешанные модульные системы

## Ключевые улучшения

### ES6+ Классы
```javascript
// Before (функция-конструктор)
function Product(id, name, price) { ... }

// After (класс)
class Product { 
  constructor(id, name, price) { ... }
}
```

### Приватные поля
```javascript
class Product {
  #id;        // приватное поле
  #name;      // доступно только внутри класса
  #price;
  
  get price() { return this.#price; }  // публичный getter
}
```

### ES6 Modules
```javascript
// Instead of require/module.exports
import { Cart, Product } from './core.mjs';
import { formatPrice } from './utils.mjs';
```

### Шаблонные литералы
```javascript
// Before
console.log("Total: $" + total);

// After
console.log(`Total: ${formatPrice(total)}`);
```

- Правильное форматирование цены
